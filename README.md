## note-wechat-app
微信小程序全栈项目

### 项目介绍
> 1.首页note 采用百度AI语音识别 把语音转成文字  
> 2.图片识别页面 采用百度AI文字识别 把图片中的文字提取出来  
> 3.我的页面 涉及小程序登录 服务端登录生成token等
